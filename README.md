# RouteRefund upload package

Upload these files to the GitHub repository root:

- CNAME
- index.html
- signup.html
- login.html
- dashboard.html
- owner-login.html
- owner.html
- styles.css
- app.js

Keep `CNAME` exactly as `routerefund.com` so GitHub Pages keeps the custom domain.

Important: this is a static MVP. It has separate pages and working buttons using browser localStorage. For real production accounts, connect Supabase Auth + database and protect owner routes server-side.
